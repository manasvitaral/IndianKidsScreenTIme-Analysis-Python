# 📱 Indian Kids Screen Time Analysis

## 📌 Project Title / Headline

**Indian Kids Screen Time Analysis: Understanding Digital Usage Patterns, Health Impacts, and Behavioral Trends**

An exploratory data analysis (EDA) project that investigates screen-time habits among Indian children and teenagers. The analysis explores relationships between age, gender, device usage, location, educational vs recreational screen usage, and potential health impacts using Python-based visualizations.

---

# 📖 Short Description / Purpose

With the increasing use of smartphones, tablets, computers, and digital learning platforms, children's screen exposure has become a growing concern for parents, educators, and healthcare professionals.

This project analyzes screen-time data of Indian children to identify usage patterns, understand factors contributing to excessive screen exposure, and examine potential health impacts associated with prolonged digital device usage.

The study leverages Exploratory Data Analysis (EDA) techniques to uncover meaningful insights that can support healthier digital habits and informed decision-making.

---

# 📂 Data Source

The dataset was obtained from Kaggle and contains information on screen-time behavior among Indian children and teenagers. It includes demographic details, device usage patterns, average daily screen time, educational versus recreational usage ratios, health impacts, and geographical classification (urban/rural).

### Dataset Features

* Age
* Gender
* Average Daily Screen Time (Hours)
* Primary Device Used
* Educational-to-Recreational Ratio
* Exceeded Recommended Screen-Time Limit
* Health Impacts
* Urban or Rural Location

The dataset was used to perform Exploratory Data Analysis (EDA) to identify screen-time trends, behavioral patterns, device preferences, and potential health-related concerns among children and adolescents.

**Source:** Kaggle – [*Indian Kids Screen Time Dataset*](https://www.kaggle.com/datasets/ankushpanday2/indian-kids-screentime-2025)

# 📊 Features & Highlights

## 1️⃣ Business Problem

The rapid growth of digital technology has increased children's exposure to screens through smartphones, tablets, computers, gaming devices, and online learning platforms.

Excessive screen time can contribute to:

* Eye strain
* Sleep disturbances
* Reduced physical activity
* Behavioral issues
* Poor work-life balance for students

However, understanding how screen usage varies across demographics and environments remains challenging.

Organizations, schools, healthcare professionals, and parents need data-driven insights to better understand children's digital consumption patterns and associated risks.

---

## 2️⃣ Goal of the Analysis

The project aims to:

* Analyze screen-time behavior among Indian children.
* Identify age groups with the highest screen exposure.
* Examine differences in screen time across genders.
* Understand preferred digital devices.
* Evaluate compliance with recommended screen-time limits.
* Investigate relationships between screen time and reported health impacts.
* Compare screen usage patterns in urban and rural populations.
* Explore the balance between educational and recreational device usage.

# 📈 Business Impact & Key Insights

### Key Insights Discovered

✅ A significant proportion of children exceed recommended daily screen-time limits.

✅ Screen-time patterns vary across different age groups.

✅ Certain device categories are more strongly associated with excessive usage.

✅ Urban and rural populations show distinct screen-consumption behaviors.

✅ Educational and recreational screen usage are not evenly distributed among participants.

✅ Higher screen-time ranges appear more frequently alongside reported health concerns.

✅ Device preferences evolve as children grow older, reflecting changing digital habits.
